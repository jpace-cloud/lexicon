# About Lexicon

Lexicon is the design system for [The Pace](https://thepace.io). It provides the shared visual language -- tokens, components, and patterns -- used across all Pace products.

## What it is

Lexicon is a three-layer design token system paired with 10 accessible React components, a Tailwind CSS preset, and plain CSS custom properties. It is designed to be the single source of truth for colour, typography, spacing, elevation, motion, and component styling.

The system is intentionally small and opinionated. Ten primitive components cover the most common UI needs. Every value traces back to a token. Every component is accessible by default.

## Who built it

Lexicon is built and maintained by [The Pace](https://thepace.io), a product studio focused on developer tools and content infrastructure. The Pace builds products including [Parlance](https://parlance.business) (content contracts and accessibility tooling) and related platform services.

## The name

*Lexicon* and *Parlance* are companion names. A **lexicon** is a vocabulary -- the set of words and their definitions. A **parlance** is the way those words are used in practice. Lexicon defines the design vocabulary; Parlance puts it to work in real products.
