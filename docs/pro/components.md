# Advanced Components <Badge type="warning" text="Pro" />

These components are available with a [Lexicon Pro](/pro/) licence. They are built on the same token system and follow the same accessibility standards as the 10 free primitives.

## DataTable

A full-featured data table with column sorting, multi-column filtering, row selection, pagination, and bulk actions. Built on top of the primitive Table component.

## Chart wrappers

Lightweight wrappers around a charting library, pre-styled with Lexicon tokens. Includes bar, line, area, and pie chart variants with consistent colour palettes and typography.

## Dashboard layout

A responsive layout shell with a collapsible sidebar, top header bar, breadcrumb navigation, and a scrollable content area. Designed for admin panels and internal tools.

## Multi-step form

A form wizard with a visual progress indicator, step validation, and animated transitions between steps. Supports linear and branching flows.

## Command palette

A keyboard-driven search overlay (Cmd+K / Ctrl+K) for navigating pages, running actions, and searching content. Includes fuzzy matching and grouped results.

## Kanban board

A drag-and-drop board with configurable columns, card templates, and swimlanes. Suitable for project management and workflow tracking interfaces.

## Calendar

A date-aware calendar component with month, week, and day views. Supports event rendering, drag-to-reschedule, and date range selection.

## File uploader

A drag-and-drop file upload zone with progress indicators, file type validation, size limits, and thumbnail previews for images.

## Rich text editor shell

A toolbar and content area shell for rich text editing. Provides the chrome (toolbar buttons, formatting menus) while allowing you to plug in your own editor engine (ProseMirror, TipTap, Slate, etc.).
