## Advanced components

Pro ships additional compound components built on the same token system:

- DataTable with sorting, filtering, and pagination
- Chart wrappers (bar, line, area, pie)
- Dashboard layout with sidebar, header, and content areas
- Multi-step form with progress indicator
- Command palette (Cmd+K)
- Kanban board with drag-and-drop
- Calendar with event support
- File uploader with drag-and-drop and progress
- Rich text editor shell

See [Advanced components](/pro/components) for the full list.
