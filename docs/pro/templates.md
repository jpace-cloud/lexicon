# Page Templates <Badge type="warning" text="Pro" />

Page templates are production-ready layouts available with a [Lexicon Pro](/pro/) licence. Each template is built from Lexicon primitives and tokens, fully responsive, and ships with both React and plain HTML versions.

::: info Planned for v1.1
Page templates are in active development. Subscribe for updates at [hello@thepace.io](mailto:hello@thepace.io).
:::

## Dashboard

An analytics overview page with a stat row, chart panels, recent activity feed, and a data table. Includes the Dashboard layout shell with sidebar navigation.

## Settings

A tabbed settings page with form sections, toggle groups, and action buttons. Uses the Tabs component for navigation and the Card component for grouping related controls.

## Onboarding

A multi-step onboarding flow with a progress bar, form fields, file upload, and a completion screen. Built on the Multi-step form component.

## Auth pages

A set of authentication screens:

- **Sign in** -- email/password form with "forgot password" link and social login buttons.
- **Sign up** -- registration form with password strength indicator.
- **Forgot password** -- email input with confirmation message.

All pages are centred on a minimal background with the Lexicon purple glow accent.

## Pricing page

A three-tier pricing layout with toggle between monthly and annual billing. Each tier is a Card with feature list and CTA button. The recommended tier is visually emphasised with the elevated variant and a purple border.

## Marketing landing

A full marketing page with:

- Hero section with headline, subhead, CTA buttons, and a purple glow background.
- Feature grid using Cards.
- Social proof / logo bar.
- FAQ accordion.
- Footer with navigation columns.
